# Datasets Base – Actividad ML2

Este directorio contiene los **datasets base** utilizados en la **Actividad ML2**, orientada al análisis de incendios forestales mediante técnicas de **aprendizaje automático (Machine Learning)**.

Los datos han sido organizados de forma modular para facilitar su uso, mantenimiento y reutilización en distintas etapas del análisis y modelamiento.
